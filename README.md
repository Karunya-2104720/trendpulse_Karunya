# trendpulse_Karunya
Project Overview

This project collects trending stories from the Hacker News API, categorizes them based on keywords, and saves the data into a JSON file.

Features
Fetches top stories from Hacker News API
Extracts required fields:
post_id
title
category
score
num_comments
author
collected_at
Categorizes stories into:
Technology
World News
Sports
Science
Entertainment
Saves data into a JSON file
Handles API errors gracefully
🛠️ Technologies Used
Python
requests library
JSON
OS module
